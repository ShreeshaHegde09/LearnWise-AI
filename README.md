# AI-Powered Intelligent Learning System

## Project Overview
An adaptive learning system that uses emotion detection, screen monitoring, and AI to create personalized learning experiences.

## Features
- Real-time emotion detection using facial analysis
- Screen monitoring via Chrome extension
- Dynamic content generation with LLMs
- Progressive learning structure
- Context-aware AI chatbot
- Reinforcement learning for user adaptation
- Quiz, summary, and flashcard generation

## Tech Stack
- Backend: Python Flask/FastAPI
- Frontend: React/Next.js
- AI/ML: TensorFlow, OpenAI API
- Database: SQLite/PostgreSQL
- Chrome Extension: JavaScript
